AI Voice Input

A Telegram bot for field sales reps: send a voice message after a customer visit, AI automatically extracts and saves customer data to the database.

GitHub Repo Get Started
⚡ How It Works
🎤

Sales rep sends a voice message

Sent directly via Telegram right after a field visit or customer meeting

🔊

Whisper converts speech → text

Runs locally with faster-whisper — no API cost, works offline

🤖

Claude AI extracts structured data

Parses the transcript and pulls out customer name, phone, status, and notes

💾

Saved to SQLite

Record inserted into phone_call table, bot replies with a confirmation

🛠 Tech Stack
✈️

python-telegram-bot

Framework for receiving and handling Telegram messages

🔉

faster-whisper

Offline speech-to-text with strong Vietnamese support

✨

Claude Haiku

AI that extracts structured fields from the transcript as JSON

🗄️

SQLite

Lightweight database, zero setup, built into Python

🎞️

ffmpeg

Converts audio from OGG to WAV for Whisper processing

🚀 Getting Started
# 1. Clone the repo
git clone https://github.com/naomitruong/ai-voice-input.git
cd ai-voice-input

# 2. Create a virtual environment
python -m venv venv
source venv/bin/activate

# 3. Install dependencies
pip install -r requirements.txt
pip install faster-whisper

# 4. Create .env file
TELEGRAM_TOKEN=your_bot_token
ANTHROPIC_API_KEY=your_api_key

# 5. Run the bot
python bot.py
